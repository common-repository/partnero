== Description ==

Partnero for WooCommerce

Launch a fully-customized affiliate program to increase customer acquisition and retention.

Partnero is a comprehensive partnership management tool that allows you to boost your store revenue by launching a bespoke affiliate program with a focus on partner experience.

Seamlessly integrate Partnero with WordPress, WooCommerce, and WooCommerce Subscriptions. Start using affiliate marketing to grow your business within minutes.

With Partnero, you can fully customize program settings: adjust commission rates, type & period, cookie lifetime and referral links. Track program performance, manage partner accounts, set individual commission rates, and provide your partners with a white-labeled partner portal.

Key features:

- Unlimited affiliates and transactions

- White-label partner portal

- Automated payouts

- Automated conversion tracking

- Fully-customizable reward setup

Explore our complete feature set [here](https://www.partnero.com/features/affiliate).

Please visit our [knowledge base](https://help.partnero.com/) or [developer documentation](https://developers.partnero.com/) to find detailed how-to guides.

== Installation ==

1. Upload plugin files to the `/wp-content/plugins/partnero` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Partnero' menu and enter Partnero API Key that can be generated from Integration section over at Partnero.
